# Overview

`ai-building-blocks`는 재사용 가능한 AI 기능을 기능별 모듈로 정리하는 개인용 모노레포입니다.

## 핵심 원칙

- 기능별 모듈을 독립적으로 유지합니다.
- 공통 로직은 `shared/`로 모읍니다.
- 각 모듈은 작은 실행 단위로 시작합니다.
- 실제 데이터와 모델은 Git에서 제외합니다.

## 기대 효과

- 새 프로젝트를 시작할 때 기존 모듈을 바로 복사하거나 연결할 수 있습니다.
- 기능 실험과 서비스화 단계를 같은 규칙 안에서 관리할 수 있습니다.
- 여러 AI 기능을 조합하는 예제를 빠르게 만들 수 있습니다.
